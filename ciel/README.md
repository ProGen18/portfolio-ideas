# 3D Personal Portfolio

A customizable 3D portfolio template built with Next.js, React Three Fiber, and GSAP.

## Personalization
All personalization data (name, SEO, social links, projects, work experience) is centralized in `app/config.ts`.
You can modify this file to make the portfolio your own.

## Tech Stack
- Next.js
- React
- React-three-fiber
- DREI
- GSAP
- Zustand
- Tailwind
